# Interactive Travel Itinerary Builder

An AI-powered travel planning assistant that helps users generate personalized multi-day itineraries based on your input — including destination, budget, and duration.

---
## Tech Stack

| Component        | Technology             |
|------------------|------------------------|
| LLM Backend      | Groq SDK via LangChain |
| RAG Framework    | LangChain              |
| Embeddings       | HuggingFace or OpenAI  |
| Vector Store     | FAISS / Chroma         |
| Scraping         | BeautifulSoup          |
| Frontend (Optional) | Streamlit           |
| Language         | Python                 |
